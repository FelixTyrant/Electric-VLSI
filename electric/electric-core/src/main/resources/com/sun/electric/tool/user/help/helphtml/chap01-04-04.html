1. Start NetBeans 6.9 or later.
  2. Register the java.net Team Server in the NetBeans IDE:
    1. Choose Window / Services in the main menu to open the Services window.
    2. Right-click the Team Servers node and choose Add Team Server.
    3. Set the Name to "java.net" and the URL to "https://java.net". Click Add.
    4. Choose Team / Team Server / Login from the main menu to log in to the server. Alternatively, you can right-click the name of the server under the Team Servers node and choose Login.
  3. Download Electric Sources from java.net .
    1. Choose File / Open Team Project... from the main menu.
    2. Search for the Electric project
    3. Select Electric: VLSI Design System and click Open From Team Server
    4. Expand the Electric: VLSI Design System node in the Team tab
    5. Expand the Sources subnode
    6. Click on Source Code Repository (get)
    7. Either enter Folder To Get in Get Sources From Team Server dialog or click the Browse button near it. The Folder to Get of the latest Electric sources is trunk/electric
    8. Choose Local Folder in the Get Sources From Team Server to download the Electric Sources. The default is ~/NetBeansProjects/electric~svn
    9. Click Get From Team Server/
    10. The "Checkout Completed" dialog will say that 7 projects were checked-out. It will suggest that you open a project.
    11. Click Open Project...
    12. Choose electric and click Open.
  4. Build Electric
    1. Right-click the electric/ node in the Projects tab.
    2. Choose Build.
  5. Open the electric-core subproject and run Electric.
    1. Expand the electric node in the Projects tab.
    2. Expand the Modules/ subnode.
    3. Double-click the electric-core subnode.
    4. The electric-core node will appear in the Projects tab. Right-click on it and choose Set as Main Project
    5. Choose either Run / Run Project (electric-core) or Debug / Debug Project (electric-core) from the main menu.
  6. Create a shortcut to start Electric from the Desktop
    1. Create a shortcut to Local Folder/electric/electric-distribution/electric-${project.version}-app/bin/electric in Unux or to Local Folder/electric/electric-distribution/electric-${project.version}-app/bin/electric.bat in Windows.
    2. Launch Electric with this shortcut.
  7. Create an Electric distribution for your organization (optional).
    1. Open the subproject electric-distribution by double-clicking the node lectric|Modules|electric-distribution
    2. Set electric-distribution as the Main Project.
    3. Select the combo box in the tool bar that shows default-config and choose distro config.
    4. Build an electric-distribution project.
    5. Copy the project to a shared location in your file system, any of:
      • directory Local Folder/electric/electric-distribution/electric-${project.version}-distro
      • JAR file Local Folder/electric/electric-distribution/electric-${project.version}-bin-jar

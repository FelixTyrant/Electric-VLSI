В Electric существует концепция текущей технологии, отображаемой в строке состояния. Эта технология используется во многих случаях, включая выор узлов и дуг в меню компонент. Существует ряд способов изменить текущую технологию, как ручных, так и автоматических.

Можно сменить текущую технологию, выбирая ее из заклядки сверху левого прямоугольника (панели "Components" или "Layers"). Electric автоматически переключает текущую технологию так, чтобы она соответствовала редактируемой ячейке. Если есть несколько ячеек, редактируемых из разных технологий, такое переключение может сильно мешать. А чтобы выключить это автоматическое переключение технологий, используйте панель "New Nodes" (в меню File / Preferences..., раздел "General" , панель "New Nodes"), и отмените выделение "Switch technology to match current cell".

Чтобы увидеть список примитивных узлов и дуг в текущей технологии, используйте команду Describe this Technology (в меню Edit / Technology Specific). Чтобы посмотреть детальное описание текущей технологии, используйте команду Document Current Technology.

Панель "Technology" (в меню File / Preferences..., раздел "Technology", панель "Technology") позволяет управлять многими общими и специфическими настройками технологии.

Можно сменить технологию по умолчанию с помощью закладки в верхнем левом углу этого диалогового окна. Эта технология по умолчанию используется, когда Electric запускается в первый раз. Она также используется при чтении старых библиотек, которые не имеют некторых параметров современных технологий.

Кроме запоминания стартовой технологии, Electric также запоминает технологию "топологии", используемую по умолчанию. Это технология с реальной геометрией (технология интегральной схемы, а не схемотехническая (schematics) или художественная (artwork)). Технология топологии по умолчанию используется для получения дальнейшей информации о схемотехнических компонентах. Она указана в разделе "Schematics" в разделе "Use scale values from this technology".

Некоторые технологии имеют переназначаемые параметры, которые в дальнейшем их определяют. Диалоговое окно "Technology" позволяет управлять этими параметрами. Дальнейшая информация об этом диалоговом окне доступна из разделов о конкретной технологии MOSIS CMOS (Раздел 7-4-2), Schematics (Раздел 7-5-1 и Раздел 3-11-2), и Artwork (Раздел 7-6-1).

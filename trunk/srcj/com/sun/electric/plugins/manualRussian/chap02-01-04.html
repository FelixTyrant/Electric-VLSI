Как только выбор сделан, Вы можете сохранить его с помощью команды Push Selection (в меню Edit / Select). Подсвечивание не изменится, но сохранится в стеке. Чтобы восстанавить этот выбор позднее, используйте команду Pop Selection.

Есть некоторое настройки выбора, которые можно установить с помощью настроек "Selection" (в меню File / Preferences ..., раздел "General", вкладка "Selection"). "Easy selection" управляет возможностью выбора объектов простым кликом, либо с помощью дополнительных действий. Вы можете выбрать, чтобы все отдельные ячейки были немедленно выбраны и чтобы весь текст аннотации был немедленно выбран. См. (следующий раздел)..

" Dragging must enclose entire object ", требует, чтобы область выбора полностью включала объект, для того чтобы выбрать его. По умолчанию, любой объект касающийся области будет выбран.

Для предотвращения случайного перемещения объекта после выбора, перемещение объекта блокируется в течение короткого времени после щелчка . Этой задержкой можно управлять. 

Когда курсор передвигается по схеме, показывется то, что будет выбрано при следующем клике. Усовершенствованный предварительный просмотр показывает в различных цветах что фактически подсвечивается (первоначально синий, но это может быть изменено параметром "Color", см. Раздел 4-6-2). Эту особенность называют "mouse-over highlighting". Если Вы не хотите видеть этот предварительный просмотр, отключите, " Enable mouse-over highlighting ".

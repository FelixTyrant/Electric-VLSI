Electric plug-ins это добавочный код, используемый для повышения функциональности системы, который можно скачать отдельно от основного кода. На данный момент доступны следующие плагины:
  • IRSIM. IRSIM это симулятор предоставленный Станфордским Университетом. Первоначально написанный на C, он был переписан на Java поэтому Electric может его использовать. Текущую версию Electric можно скачать с Static Free Software www.staticfreesoft.com/electricIRSIM-8.02.jar.
  • Bean Shell. Используется для осуществления обработки всех параметров. Для выполнения практически всех операций в Electric требуется этот plug-in. Он также доступен на www.beanshell.org.
  • 3D. 3D компонент позволяет отображать ИМС в трехмерном пространстве. В добавок, к 3D плагину можно установить контролер (3D ось). 3D пакет доступен для скачивания по адресу www.j3d.org. Контролер 3D оси можно скачать с Static Free Software www.staticfreesoft.com/electricJava3D-8.02.jar
  • Animation 3D анимация возможна при условии наличия 3D компонентов, описанных выше, а так же при наличии Java Media Framework (JMF) анимационного кода. Java Media Framework доступен по адресу Sun Microsystems java.sun.com/products/java-media/jmf анимационный код по адресу Static Free Software www.staticfreesoft.com/electricJMF-8.02.jar.

Для корректной работы плагинов они должны быть добавлены к основному коду. Например, для добавления beanshell (имя файла "bsh-2.0b1.jar"), наберите:

java -classpath electric.jar:bsh-2.0b1.jar com.sun.electric.Launcher

Под Windows, вы должны разделять jar файлы с помощью ";"

java -classpath "electric.jar;bsh-2.0b1.jar" com.sun.electric.Launcher

Запомните, вы должны однозначно ссылаться на главный класс Electric (com.sun.electric.Launcher), так как при использовании плагинов все jar файлы группируются в один, так называемый, "classpath".

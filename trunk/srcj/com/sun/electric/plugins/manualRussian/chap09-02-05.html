Используя инкрементальное или иерархическое проверочное устройство, мы можем обнаружить четыре типа ошибок. Ошибка расстояния (Spacing) причиной такой ошибки является геометрические неточности, например, два элемента находятся очень близко, но не подсоединины. Notch также ошибка относящиеся к расстоянию между элементами, но в данном случае элементы присоединены. Ошибка размера (Minimum size) возникает, если в одной рабочей области обнаруживается элемент, чьи размеры не сопоставимы с размерами остальных элементов. Ошибки разрешающей способности (Resolution), возникают при использование элементов, чьи размеры меньше чем установленная разрешающая способность.

В добавок, для проверки нарушений построения, проверочное устройств использует информацию о связанности элементов. Использование такой информации может помочь проектировщику исправить некоторые ошибки( под связанностью подразумевается физическая связь элементов). Например, если два наложенных друг на друга узла не соединены дугой, то при проверке это вызовет ошибку, хотя с точки зрения геометрии построения все верно. For example, if two overlapping nodes are not joined by an arc, Это происходит потому, что проверочное устройство, считав информацию о связанности, знает, что на схеме подсоединено, а что нет.

При запуске DRC, устройство обрабатывает всю рабочую область. Иногда требуется проверить только какой-то определенный участок схемы, игнорируя ошибки остальной области. Для этого используется "cloaking" слой, область покрытая этим слоем, при DRC проверки будет игнорироваться. Для нанесения cloaking слоя, из компонентного меню "Misc." выберите элемент "DRC Exclusion" и отметьте область игнорирования DRC. При запуски проверки отобразятся ошибки только непокрытых элементов.

Настройки "Правил проектирования" можно найти на закладке меню) File / Preferences..., "Technology", ""Design Rules" ). В настройках вы можете посмотреть и изменить придел размерности для текущей технологии. Диалоговое окно представляет возможность корректировать размеры как узлов так и слоев ("Node Rules" "Layer Rules"). В левой части окна находится список узлов и слоев чьи размеры можно менять, справа их текущие размеры(size) или текущее расстояние(distance) от подложки.

Для узлов в данной технологии вы можете указать только минимальные геометрические размеры.

Для слоя, помимо геометрических размеров, мы можем указать площадь слоя и площадь покрытия для каждого слоя. Так же можно установить размер промежуточного слоя, для этого отметьте пункт "Show only 'to' entries with rules", который оставит только те слои для которых это возможно.

Настройка межслоевого пространства может быть в трех формах: normal, wide, and multicut. Normal и multicut настройки могут быть как для подсоединенных слоев так и для не подсоеденненых. Подсоединненые слои это те слои по которым идёт ток, а не подсоединенные соответственно по которым ток не идет.

Настройка wide rules добавляется при выбор слоя с большими геометрическими размерами.

  • Кнопка "Factory Reset" означает сброс всех изменений и возвращение к первоначальным настройкам.
  • "Foundry" это выбор фундаментальной технологии.
  • Окно "Min resolution" показывает минимальное воспроизводимое разрешение. При нуле проверка по разрешению не проводится. При проверке разрешения, все геометрические размеры равные установленному значению или меньшие его, будут помечаться как ошибка. Например, в текущей технологии MOSIS правила требуют чтобы границы были в четверть лямбда или меньше, поэтому значение .25 в этом поле обнаружит такую ошибку.

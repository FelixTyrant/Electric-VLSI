Хотя расстояния по умолчанию описываются в безразмерных единицах, они могут быть выражены в реальных единицах измерения. Вы можете выбрать реальные единицы измерения для различных величин в пункте ⌠Units■ в меню File / Preferences... (раздел "Technology", панель "Units").

Часть этого диалогового окна в этой версии недоступна, т.к. система не использует эту информацию.

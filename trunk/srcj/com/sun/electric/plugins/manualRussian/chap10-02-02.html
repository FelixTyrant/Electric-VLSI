После строки заголовка должны быть объявлены все внешние библиотеки и экспорты. Это позволяет программе чтения быстро находить все библиотеки которые понадобятся для проектирования, и восстанавливать некоторые недостающие ячейки и экспорты. Ячейки перечисляются ниже их библиотек. Экспорты перечисляются ниже их ячеек Если есть многочисленные внешние строки библиотек, они сортируются по названию библиотек; когда есть многочисленные внешние ячейки в библиотеке, они сортируются по названию; и там где есть многочисленные внешние экспорты в ячейке, они сортируются по их названию

Систанксис внешней библиотечной ссылки:

L<name> | <path>
<name>название внешней библиотеки.
<path>полный путь к файлу на диске с библиотекой.

Название библиотеки используется в JELIB-файле для ссылки на эту библиотеку. Фактическое название этой библиотеки получается из пути.

Синтаксис ссылки на внешнюю ячейку:

R<name> | <lowX> | <highX> | <lowY> | <highY> | <creation> | <revision>
<name>название внешней ячейки.
<lowX>нижний предел содержимого ячейки по оси Х.
<highX>верхний предел содержимого ячейки по оси Х.
<lowY>нижний предел содержимого ячейки по оси Y.
<highY>верхний предел содержимого ячейки по оси Y.
<creation>дата создания ячейки (Java формат).
<revision>дата изменения ячейки (Java формат).

Java формат для дат (даты создания и изменения) - это миллисекунды с момента начала нашей эры (полночь 1 января, 1970, GMT).

Синтаксис внешней ссылки на экспорт:

F<name> | <centerX> | <centerY>
<name>название внешнего экспорта.
<centerX>координана по оси X цетнра многоугольника экспорта.
<centerY>координана по оси Y цетнра многоугольника экспорта.

Примеры:

    Lspiceparts|/home/strubin/electric/spiceparts.jelib
    Rgate;1{sch}|-4|4|0|2|1092185029000|1092185060000
    Fout|0|2

Объявляет что внешняя библиотека, называемая "spiceparts" будет использоваться текущей библиотекой, и что она может быть найдена в "/home/strubin/electric/spiceparts.jelib". В этой библиотеке есть ячейка, названная "gate;1{sch}", чье содержимое находится от -4 до 4 по оси X и от 0 до 2 по оси Y/ В этой ячейке есть экспорт, названный "out" с центром в (0,2).

FastHenry это инструмент анализа индуктивности (Jacob White). При генерации FastHenry записи создаётся список подмножества дуг для данной ячейки. Для добавления дуги к записи, выберите дугу и используйте команду Fast Henry Arc Properties... меню (Tools / Simulation (Others) ).
В диалоговом окне представлены факторы отмеченной дуги. Наиболее важный пункт: "Include this arc in FastHenry analysis". Отметьтив этот пункт, дуг будет включена в запись. При включение этого пункта активизируются остальные пункты. Вы можете установить толщину дуги (если пользователь н устанавливает свое значение, то используется дефолтовое значение) Также можно установить разделение дуги по ширине и длине. Расстояние между двумя точками можно конкретизировать с помощь определения координат точек.
Дуги можно разделить по группам, для этого нажать "New Group", после чего перераспределить дуги по группам.

После того как вы перебрали все дуги, можно приступать к созданию FastHenry записи. Для генерации используйте команду Write FastHenry Deck... меню (Tools / Simulation (Others)). Как и для прочих инструментах FastHenry имеет набор настроек, который можно посмотреть в "FastHenry" свойствах меню (File / Preferences..., "Tools" section, "FastHenry" tab).

Диалоговое окно представляет возможность установить тип анализа по частоте (с постоянной частотой или последовательностью с начальной, конечной тачками и отставанием по фазе) Можно установит единичный или множественный анализ (при проведение множественного анализа можно определить количество проведения анализа ) С помощью данных настроек вы можете установить определенные опции для каждой дуги. Например, толщину и количество подмножеств по длине и ширине.

A library является коллекцией элементов, которая формирует иерархию. Предписывая эту последовательность в память Electric-полная библиотека в одном файле на диске, который читается или написан. Это нужно, чтобы cоздавать множественные библиотеки в Electric Когда есть множественные библиотеки, Вы можете коммутировать между ними с Change Current Library... командой (в меню File) или используя контекстное меню библиотеки в проводнике элемента. (см. Section 4-8). Чтобы видеть,какая библиотека используется - List Libraries команда.

Для создания новой,очистка библиотеки используйте New Library... (в меню File). Для изменения имени текущей библиотеки,используйте Rename Library.... Удалить библиотеку: Close Library команда. Она удаляет только изображение запоминающего устройства, не файл на диске. Обратите внимание, что библиотечные изменения слишком обширны.

Возможно связать две библиотеки, помещая образец элемента от одной библиотеки в другой (с помощью Place Cell Instance... команда в меню Cell). Когда это случается, библиотека с образцом (основная библиотека) связана с библиотекой с фактическим элементом ( reference library). Справочная библиотека необходима, чтобы работать с основной библиотекой, она будет читаться всякий раз, когда основная библиотека читается.

Если библиотеки, с которыми работают- независимы, возможно, что ссылка к элементу в другой библиотеке не будет соответствовать фактическому элементу в той библиотеке. Тогда, Electric создает элемент "placeholder", который соответствует первоначальным{оригинальным} техническим требованиям. Таким образом, ссылки к библиотеке нет, потому что элемента там нет .Просмотр списка всех элементов "placeholder", которые были созданы из-за таких проблем, использование General Cell Lists... команды ( Cell / Cell Info) и выбрать "Only placeholder cells".

При чтении и записи библиотек, Electric предлогает множество способов выбирать каталог в файловой системе. Выборы доступны в поле "Working directory" "General" (в меню File / Preferences..., "General" section, "General" tab):

  • "Based on OS " указывает, что Electric должен выбрать каталог, основанный на стандартных настройках, используемых в вашей операционной системе.
  • "Use current directory" указывает, что Electric должен использовать текущий каталог.
  • "Use last used directory" должен запомнить последний каталог, который использовался.

Масштаб можно несколькими способами. Команда Zoom In в меню (Window), даёт изображение крупным планом. Команда Zoom Out соответственно уменьшение масштаба.

Также масштаб можно изменять, удерживая shift плюс правый клик мыши. (see Section 1-8). Обратная операция выполняется также с удерживанием shift-right плюс выделение правой кнопкой мыши нужной области.

Самый простой способ это использование соответствующей команды на панели инструментов. Масштабирование еще осуществляется с помощью команды Toggle Zoom меню (Edit / Modes / Edit).

Также полезна команда Fill Window в меню ( Window), которая увеличивает текущий объект на весь экран.

Для увеличения отдельного участка изображения используется команда Focus on Highlighted меню (Window / Special Zoom). Для масштабирования элементов без связей используйте команды. (see Section 2-1-3). Также можно использовать команду Zoom Box, которая позволяет сначала растянуть участок, а затем его увеличить.

Последняя команда Match Other Window, которая перерисовывает объект в соответствии с масштабом остальных объектов.

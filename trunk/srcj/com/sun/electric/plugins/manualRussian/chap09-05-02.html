В дистрибутив Electric включен внутренний симулятор вентильного уровня - ALS. Способный обрабатывать схемотехнический и топологический уровень, а также распознавать язык описания аппаратуры VHDL. Симулятор работает с транзисторами выполненными по технологии MOS и с некоторыми цифровыми логическими вентилями. Он также может быть дополнен функциональным описанием любой схемы, с помощью "hardware description language" описание которого приводится в данном разделе.

В качестве примера, загрузите библиотеку "samples" и просимулируйте ячейку "tool-SimulateALS{sch}". Для загрузки библиотеки воспользуйтесь командой Load Library из меню Help / Samples.

Для симуляции ячейки в текущем окне используйте команду ALS: Simulate This Cell из меню Tools / Simulation (Built-in). После выполнения команды появится окно симуляции для дальнейшего контроля действий. (подробнее об окнах симуляции Section 4-12-1). Так как ALS является внутренним симулятором, пока запущен ALS "движок", вы можете подавать управляющие воздействия и незамедлительно получать результат.

ALS способен работать с транзисторами переменной мощности. Для установления использования маломощных транзисторов в меню Tool / Simulation (Verilog) / Transistor Strength выберите команду Weak. Если вы хотите использовать транзисторы нормальной мощности, то в том же меню выберите команду Normal. Выбор мощности транзистора необходимо производить непосредственно до начала симуляции.

Настройки

Настройки "Simulators" из меню File / Preferences..., "Tools" закладка, "Simulators", содержат несколько пунктов влияющих на работу ALS симулятора. Установка пункта "Multistate display" приводит к тому что симулятор показывает форму сигнала различными цветами для разграничения транзисторов по мощности. Соответственно если проигнорить этот пункт, то все сигналы будут отображаться одним цветом. Остальные наиболее важные настройки уже обсуждались в главе 4: ещё разочек:).

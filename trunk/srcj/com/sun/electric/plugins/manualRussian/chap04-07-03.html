Часто требуется упорядочить несколько объектов, для этого используют команды из Edit / Move.

Align Horizontally to Left выравнивание по Y слева, Align Horizontally to Right выравнивание по Y справа, Align Horizontally to Center выравнивание по Y по центру, Align Vertically to Top выравнивание по X сверху, Align Vertically to Bottom выравнивание по X снизу, Align Vertically to Center выравнивание по X по центру.

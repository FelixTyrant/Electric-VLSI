Мaze трассировщик обрабатывает неразведенные дуги в соответствие с оригенальной трассировкой. Для запуска воспользуйтель командой Maze Route меню Tool / Routing. Цепь отмеченная при запуске трассировки будет обработана. При этом разводятся все неотработанные дуги.

Обратите вниание, что если трассировщик не находит путь, то трассировка заканчивается. Это делается для дитального контроля за процессом трассировки.

Трассировщик так же пытается развести узлы, по наименьшему расстоянию. Чем больше растояние тем больше времени требуется для разводки.

Для наглядности загрузите пример "tool-RoutingMaze" из Load Library. Данный пример содежит не разведенные соединения.

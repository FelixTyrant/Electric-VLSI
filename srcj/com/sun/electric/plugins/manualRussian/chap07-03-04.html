EDIF (Electronic Design Interchange Format) используется для обмена информацией между различными САПР. Текущая версия EDIF - "4 0 0", но Electric экспортирует и импортирует файлы версии "2 0 0". Прочую информацию можно узнать в разделах 3-9-2 и 3-9-3. Опции EDIF можно изменить на панели "EDIF" (в меню File / Preferences..., раздел "I/O", панель "EDIF").

Это диалоговое окно управляет способом записи EDIF файла - schematic или netlist views (по умолчанию netlist). Можно также масштабировать ввод EDIF.

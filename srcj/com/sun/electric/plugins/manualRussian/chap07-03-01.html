Electric способен считывать и записывать цепи в ряде различных форматов. Эти действия производятся с помощью команд Import и Export (в меню File). См. Section 3-9-2 for more on Import; См. Section 3-9-3 for more on Export.

Для верного управления проебразованием форматов существуют множество различных диалоговых окон для различных типов файлов (в меню File / Preferences..., раздел "I/O").

К сожалению, многие из этих форматов содержат ислючительно чистую геометрию без всякой информации о соединениях цепей. При чиении онир воспринимаются как узлы слоя (pure-layer nodes). Это означает что транзисторы, межсоединения и прочие узлы из более чем одного слоя правильно не отображаются. Хотя визуально ячейка является верной, ее нельзя проанализировать на уровне электрической цепи. Для того чтобы преобразовать такие узлы в правильные компоненты Electric, следует использовать node (см. Раздел 9-10-2).

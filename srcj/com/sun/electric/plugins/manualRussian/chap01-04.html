Запуск Electric варьируется в зависимости от платформы.
  • UNIX/Linux Запуск под UNIX/Linux может осуществлять используя, какой-либо файловый менеджер или используя консоль, набрав:
    java -jar electric.jar

  • Windows Аналогично запуску под Linux, Electric можно запускать, нажав на иконку "electric.jar", или набрав в командной строке:
    java -jar electric.jar

  • Macintosh Macintosh компьютер должен использовать OS 10.3 или более позднюю версию. Запуск осуществляется также либо нажатием на иконку "electric.jar", либо через командную строку:
    java -jar electric.jar

Если запуск не осуществился ни одним из описанных способов, попробуйте альтернативный вариант:

java -classpath electric.jar com.sun.electric.Launcher

Ниже приведены некоторые полезные опции, которые можно использовать при работе с Electric. Для запуска этих опций допишите их обозначение через пробел в конце командной строки.

  • -mdi документационный интерфейс (Electric представлен обычными окнами с маленьким окном редактирования).
  • -sdi по большому счету, тот же интерфейс, только окна более подробно классифицированы.
  • -s <script> запускаются <script> файлы байт кода.
  • -batch запуск пакетного режима (никакого юзерского интерфейса не отображается).
  • -version обеспечивается полная информационная версия, включая данные о сборке.
  • -v обеспечивается короткая информационная версия.
  • -help список всех команд.

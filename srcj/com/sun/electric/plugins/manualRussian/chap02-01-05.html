В используемой схеме, объекты могут накладываться, вызывая затруднения при выборе. Чтобы упростить выбор, объекты могут быть отмечены так, чтобы они больше не easy-to-select, что означает, что их нельзя выбрать стандартным способом.

Чтобы выбрать объекты hard-to-select, используйте команду Special Select (в меню Edit / Modes / Select). Вы можете также щелкнуть по кнопке "Special Select" на панели инструментов,чтобы осуществить специальный выбор.

Легкость выбора распространяется не только на узлы и провода. Есть четыре "класса" объектов, которые могут быть выбраны:

 Basic objects (all arcs, primitive nodes, and port names)

  • Образцы ячейки
  • Основные объекты (все провода, простые узлы, и имена портов)
  • Текст аннотации (имена и другой текст, помещенный в узлы и дуги)
  • Текст образца (нерасширенный имена образца ячейки)

По умолчанию, первые три класса easy-to-select, и текст образца - hard-to-select. Если Вы отмените выбор "Easy selection of cell instance" в диалоговом окне, то ячейки становятся hard-to-selection. Если Вы отмените выбор "Easy selection of annotation text" в диалоговом окне, то текст аннотации становится hard-to-selection.

Хотя все узлы и провода относятся к easy-to-select, Вы можете управлять ими индивидуально, отменив выбор в поле "Easy to Select" в их диалоговом окне (используя Object Properties ... команда в меню Edit / Properties). Если выбраны составные объекты, диалоговое окно Object Properties ... всплывает в нижнем правом углу, чтобы изменить опции выбора.

Существуют специальные команды в меню Selection для того, чтобы иметь дело с easy-to-selection узлами и проводами. Вы можете выделить все easy-to-selection объекты в текущей ячейке с помощью команды Select All Easy. Точно так же Вы можете выбрать те, которые не easy-to-selection с помощью команды Select All Hard. Чтобы изменить способ выбора набора объектов, подсветите их и используйте, или Make Select Easy , или Make Select Hard.

Преобразование Технологий в библиотеки

Лучший способ создать новую технологию - это измение существующей. Используйте Convert Technology to Library for Editing... команду (в меню Edit / Technology Specific) и выбирайте подобную технологию. К сожалению, Schematic и Artwork технологии слишком сложны для редактирования и не могут быть преобразованы.

Преобразование технологии в библиотеку создает библиотеку с таким же именем, как и у технологии. Заметьте, что технологии с опциями (такими как MOSIS CMOS) будут преобразованы только с их текущими настройками, а опции больше доступны не будут.

Режим редактирования технологий

Как только библиотека технологии создана, редактирование ее ячеек будет осуществляться в специальном Режиме редактирования технологий. Система понимает, что надо использовать режим редактирования технлогии, так как ячейки отмечены как "Часть библиотеки Редактора технологии" (см. Cell Properties... команду Cells меню).

Преобразование Библиотек в технологии

Для того чтобы преобразовать библиотеку технологии в технологию, используйте Convert Library to Technology... команду.

У Вас есть возможность назвать технологию, Вы можете также запросить Java код (этот код можно скомпилировать в Electric чтобы установить технологию постоянно). Если технология с тем именем, которое Вы хотите использовать, уже существует, Вы можете сделать запрос о переименовании, или можете выбрать другое имя для новой технологии (заметьте, что технологии могут быть переименованы с помощью команды Rename Current Technology...).

Если в библиотеке ошибка, преобразование отменяется и Вам дается шанс исправить библиотеку. Вообще, неправильная часть библиотеки подсвечивается. Если не обнаружено никаких ошибок в переводе, Создастся новая технология в Electric и она будет текущей.

Перед созданием любой схемы с новой технологией, желательно создать новую библиотеку (используйте команду New Library... меню File) так, чтобы испытательная схема была не сохранена библиотекой, которая описывает это.

Очистка

После нескольких периодов редактирования технологии, могут быть много библиотек и технологий. Вы можете удалить текущую технологию с помощью команды Close Library меню File (для того чтобы сделать другую библиотеку текущей, используйте команду Change Current Library... меню File).

Использование Библиотек Технологий

Как только библиотека была сформирована настолько успешно, что описала технологию, она может быть сохранена на диск с помощью команды Save Library меню File. Таким образом, при следующем запуске Electric, она может быть прочитана с диска и преобразована в технологию.

Инструмент разводки включает несколько подсистем создания маршрутов. Трассировщик использует два набора рисунков для соединения соседних ячеек. maze-router осуществляет единичную разводку. river-router также доступен и позволяет осуществлять много полосную параллельную разводку.

Некоторые из этих маршрутов используют "Unrouted Arc", "тонкие дуги", которые могут подсодинятся к двум компанентам. Создание "rats nests" этих дуг формеруют графическое описание для далнейшего использования их трассировщиками. Создание "Unrouted Arc" осуществляется с помощью Generic Technology (подробнее Section 7-6-3). Для создание используйте команду Get Unrouted Wire из меню Tool / Routing.

В итоге, команды Copy Routing Topology и Paste Routing Topology используются для создания unrouted arcs в ячейки (the "pasted" cell), где компаненты связаны с ячейкой (the "copied" cell). Далее команда Paste Routing Topology сопоставляет полученное.

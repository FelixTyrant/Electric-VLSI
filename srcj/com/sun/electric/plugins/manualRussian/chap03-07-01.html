Чтобы получать некоторую основную информацию о текущей ячейке (размер, даты, и т.д) используют Describe this Cell команда (в меню Cell / Cell Info).

Чтобы получать информацию больше чем об одной ячейке, используйте General Cell Lists... команду. Окно для выбора поднабора ячеек в текущей библиотеке.

Секция "Which cells:" выбирает элементы, которые будут перечислены (все, только используемые в других элементах, и НЕ используемые в текущем элементе, или только элементы "placeholder": созданные из-за неисправных библиотечных зависимостей, see Section 3-9-1).

Раздел "View filter:" позволяет только некоторым views быть отображенным.

Раздел "Version filter:" позволяет удаление старших или более новых вариантов элементов.

Раздел "Display ordering:" управляет порядком, в котором выбранные элементы будут перечислены.

Раздел "Destination:" позволяет Вам формировать дамп этой распечатки к файлу на диске, отформатированному для электронных таблиц.

Последние пять столбцов показывают использование элемента и четыре бита состояния{режима}. Использование - число раз, что этот элемент появляется как образец в других элементах. Биты состояния{режима}:

  • "L" если содержание элемента блокировано
  • "I" если образцы в элементе блокированы
  • "C" если элемент - часть библиотеки логических элементов
  • "D" если элемент передал проверку проектной нормы

Для большего количества информации элемента, используйте команды меню Cell / Cell Info. List Nodes in this Cell команда показывает все непримитивные точки разветвления в текущем элементе. List Cell Instances команда показывает все образцы элемента ниже текущего элемента. List Cell Usage команда просматривает иерархию и находит элементы, которые содержат текущий элемент как образец.

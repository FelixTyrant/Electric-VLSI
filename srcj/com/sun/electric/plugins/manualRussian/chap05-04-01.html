Для информационных целей возможно отображение стрелки - указателя на дугах, кторая указывает направление. Эта хакартеристика может быть изменена командой Toggle Directionality (в меню Edit / Arc). Она также может управляться из диалогового окна Object Properties... (в меню Edit / Properties).

Элементы управления в окне Object Properties... предлагают варианты размещения головной части стрелки на одном из концов дуги, обоих концах, ни на одном из концов. Это позволяет произвольно комбинировать передние части и тела стрелок для отображения произвольно сложных направленных схем.

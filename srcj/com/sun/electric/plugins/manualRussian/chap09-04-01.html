В Electric включены два собственных симулятора (описание которых будет изложено позднее), также Electric способен создавать управляющие записи для внешних симуляторов. Возможность взаимодействовать с внешними устройствами осуществляется посредством команд Simulation (Spice), Simulation (Verilog), и Simulation (Others) меню (Tool).

Дистрибутив Electric не включает описанные выше симуляторы!!!!

Electric может создавать выходные файлы для следующих симуляторов

ArchSim симулятор - экспериментальный симулятоор. Его выходная форма может быть отражена с помощью команды Display ArchSim Journal... меню (Tool / Simulation (Others)).

Многие команды доступны через использование "quick keys". Некоторые "горячие клавиши" приведены на скриншоте. Например, команда New Cell... меню (Cell) имеет "горячию клавишу" "Control-N". На Windows и UNIX системах "Ctrl-N", как не трудно догадатся, означает, что для использования горячих клавиш необходимо нажать N при этом удерживая Ctrl.

Для изменения "горячих клавиш" используйте свойства "Key Bindings" меню (File / Preferences..., "Tools" , "Key Bindings" закладка). Диалоговое окно отобразит иерархическую структуру уже зарезервированных клавиш и предоставит возможность внести изменения.

Убрать клавишу можно с помощью "Remove" button, соответственно добавить, используя "Add" button. Кнопка "Reset" установит все значения по-умолчанию. Будьте осторожны при изменении уже зарезервированных клавиш, это может осложнить работу других пользователей.

С помощью "горячих клавиш" можно получать доступ не только к каким-то конкретным командам, но и к меню команды в целом. Клавиши, используемые для этих целей, подчеркнуты(смотри скриншот). Например, к меню File можно обратится, нажав "F", а к меню Print... нажав "P". Это значит что удерживая Alt и, нажав "FP" вы можете сразу попасть на меню команд печати.

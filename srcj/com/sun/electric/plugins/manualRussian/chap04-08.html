Компонентное меню, как правило, располагается слева каждого окна. При желании его можно переместить вправо, либо командой On Right меню (Windows / Side Bar), либо исправив дефолтовые настройки "Side bar defaults to the right side" в закладке "General" (File / Preferences..., "General", "General" tab)

"Explorer"(обозреватель) отображает иерархическое дерево с тремя главными секциями: LIBRARIES(библиотеки), JOBS(тех.операция), и ERRORS(ошибки).

LIBRARIES показывает все подключенные библиотеки с элементами. Ячейки в библиотеках можно расположить тремя способами:
  • Alphabetically(алфавитный порядок)
  • By group(по группам)ячейки остаются в алфавитном порядки, плюс разбиваются на группы.
  • By hierarchy(иерархически)отображаются только ячейки наивысшей иерархии.
Для изменения отображения, надо нажать правой кнопкой мыши на LBRARIES и выбрать подходящий вид. Обратите внимание на то, что текущая библиотека подсвечивается.

В секции JOB отображаются все задачи проводимые в Electric. Обычно эта секция пустует, но если в Electric'e одновременно запустить несколько задач, то эта секция позволит манипулировать проходящими процессами.

Последняя закладка ERRORS показывает листинг ошибок, генерируемых инструментами (DRC, ERC, NCC, и т.д.).

Для каждой библиотеки и ячейки доступно свое контекстное меню.



Контекстное меню LIBRARIES состоит из трех частей. Первые три пункта предназначены для упорядочения расширения дерева. Средний пункт создаёт новую ячейку в библиотеки. Нижние три пункта, способ отображения библиотек.

Контекстное меня какой-либо конкретной библиотеки состоит из пяти пунктов. Первые три, также отвечают за упорядочение расширения дерева. Следующий пункт делает выделенную библиотеку текущей Следующий пункт добавляет библиотеку в репозиторий проекта. (подробнее Section 6-12). Далее создание новой ячейки. Последние три пункт позволяют соответственно изменять название, сохранять и закрывать библиотеку.

Контекстное меню ячейки состоит из 5 пунктов. Первые два это выбор рабочего окна. Следующие два пункта позволяют добавлять объект либо в текущую ячейку либо в новую. Далее три пункта позволяющие создать новую версию ячейки, сделать копию ячейки и удалить ячейку. Далее изменение названия и вида ячейки. Последний пункт позволяет трансформировать группу ячейки.

Контекстное меню группы ячейки состоит из двух частей. Первые три для упорядочения расширения дерева. Оставшиеся два для создания или переименования ячеек в группе.

Контекстное меню schematic ячеек состоит из двух частей. (Section 7-5-2подробней о страницах schematics). Первая часть для корректировки ячейки, вторая для до создания/удаления страницы в ячейке.



Контекстное меню процесса секции JOBS состоит из трех пунктов: "Get Info" получение дополнительной информации; "Abort" запрос на прекращение процесса(это не всегда возможно!); и "Delete" удаление процесса из списка задач.

Auto-stitching трассировщик ищет соседнии узлы. которые не явно соединенны и соответственно помещяет контакт для явного соединения. Если, например, узел имеет выводы питания и землю свеху и снизу, а порты находятся слева и справа, то целесообразно использовать auto-stitching трассировщик, для корректного соединения.

Аuto-stitching трассировщик помещяет соединение, когда выполняется следующие:

  • проетирование топологии (auto stitching в схематике не работает).
  • Порты содержатся у обоих узлов. Потомучто соединение проклядывается между портами.
  • Узлы в ячейки должны соприкосатся или частично перекрыватся, это создают неявное соединение. Когда контактная площядка узла имеет экспорт, она должна быть таких же размеров как и подсоединяемый контакт.
  • Подключение портов друг к другу недопустимо.

Запустить трассировщик можно следующе Enable Auto-Stitching команда из меню Tool / Routing. После этого трассировщик начнет свою работу. Для остановки проделайте туже операцию. Для однократного запуска трассировщика, для конкретной ячейки Auto-Stitch Now, для выделенной области команда Auto-Stitch Highlighted Now.

auto-stitcher позволяет типизировать соединения для трассировщика. В "Routing" настройках меню File / Preferences..., "Tools" закладка, "Routing" Вы найдете все возможнве свойства трассировщика.

Когда атрибуты созданы на ячейках, они становятся параметрами для той ячейки. Ячейки имеет копию атрибута, помещенного в узел, и атрибут узла может тогда быть установлен независимо от атрибута ячейки. В такой ситуации, значение атрибута узла - "фактический" параметр для ячейки, и значение атрибута ячейки - "формальное", или заданное по умолчанию значение параметра..

Один пример использования параметров ячейки находится в Spice примитивах , где определяемые пользователем значения (типа напряжения) сообщены в значок в эксплорере Spice. (cм Section 9-4-3).

Другое использование параметров ячейки должно параметризовать размер транзистора в схемном решении (или параметризовать масштабируемые транзисторы размещения в MOSIS CMOS технологий., см Section 7-4-2) Ширина транзистора и длина могут быть определены в значениях параметра, позволяя брать единственную{отдельную} ячейку в различных формах. Объединяя{комбинируя} эти параметры с интерпретирующим средством языка, произвольное математическое выражение может быть помещено в транзистор, который комбинирует{объединяет} значения параметра, чтобы формировать точный размер транзистора.

В ячейке, параметр показывают с его названием и заданным по умолчанию значением. Чтобы видеть фактическое значение от иерархии, создайте выражение Java со значением "@PNAME", где PNAME - название параметра. Удостоверьтесь, чтобы установить его Код на Java. Когда сначала определено, параметр не имеет никакого фактического значения.

Если параметр добавлен к ячейке, существующие случаи той ячейки не могут должным образом модифицироваться. Чтобы сделать это, используйте ,Update Attributes Inheritance on Node команда (в меню Edit / Properties). Чтобы сделать это везде, используйте Update Attributes Inheritance all Libraries команда.

Parameter Text

Параметры на ячейках не привязаны ни к какому узлу или дуге. Вместо этого, они с плавающей точкой свободны в ячейке. Вы можете выбрать текст и перетащить его к любому местоположению в ячейке.

Параметры ячеек помещены в то же самое местоположение, поскольку они появляются в ячейке. В схемных решениях, местоположение параметра на значке определено местоположением параметра на типовом значке, в схематической ячейке.

Если Вы не желаете видеть текст параметра , выберите текст параметра в ячейке, используйте Object Properties... команда (в меню Edit / Properties), и проверка "Invisible outside cell".

Вы можете отключить дисплей параметров на ячейках, выбирая их, и используя Hide All Attributes on Node команда (в меню Edit / Properties). Вы можете все параметры сделать отображенными на ячейке, используя , See All Attributes on Node команда. Чтобы восстанавливать заданную по умолчанию видимость параметра на ячейке, используйте Default Attribute Visibility команда.

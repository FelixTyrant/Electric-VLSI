Library Directory Management

By default Electric searches for Libraries in the working directory, absolute file path references, and Electric's LIB directory.

Users can specify additional directories to search for Libraries using a file called "LIBDIRS" placed in the working directory. The LIBDIR file specifies additional paths to search for library files.

The LIBDIR file has the following syntax: 


* <comments>
include <another_LIBDIRS_file>
<library_directory>

Paths may be absolute or relative.

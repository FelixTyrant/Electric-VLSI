Экспорт выбирается, щелкая на тексте, или щелкая на узле, от которого они экспортируются. Если очень плотный проект затрудняет экспортный выбор , Вы можете выбрать из списка, используя Select Object... команду (in menu Edit / Selection).

Чтобы видеть весь экспорт, который был определен в текущей ячейке, используйтеShow Exports команда (in menu Export). List Exports команда, дает ту же самую информацию, но в форме текста, Summarize Exports команда дает список текста, который уменьшен. Чтобы видеть список экспорта, который электрически связан с текущим объектом, на многократных уровнях иерархии, используите List Exports on Network and List Exports below Network commands (in menu Tool / Network).

Как только порт экспортировался, его характеристики могут быть изменены, выбирая экспортное название и используя Object Properties... команда (in menu Edit / Properties). You can change basic export information such as the name, characteristic, and reference name (if applicable). Вы можете изменить основную экспортную информацию типа названия, характеристики, и сослаться на название (если это применимо). Вы можете управлять экспортным состоянием например, рисуется ли это всегда, и действительно ли это появляется на отображениях{пиктограммах}. Вы можете также изменить экспорт, редактируя размер, шрифт, цвет, стиль, и оборот названия. См.Section 6-8-1

Специальные кнопки в Экспортном окне Свойств позволяют Вам исследовать связанные объекты. Кнопка "Highlight Owner" показывает узлу, на котором этот экспорт постоянно находится, и кнопка "Attributes" поднимает окно Атрибутов для экспорта (См. Section 6-8-5 про Атрибуты).

Вы можете изменить характеристики многих экспортов сразу, выбирая их и используя Object Properties... команда (in menu Edit / Properties). Этот мультиобъектный диалог предназначен "Для всего выбранного экспорта: ", который заменит все экспортные характеристики сразу. Вы можете изменить название экспорта, используя Rename Export... коанда (in menu Export).

Порты и экспорт могут быть отображены на экране многими различными способами. Чтобы управлять этим, используйте"Ports/Exports" (in menu File / Preferences..., "Display" section, "Ports/Exports" tab). Окно предлагает три варианта для портов и экспорта: "Full Names" показывают текстовые названия, " Short Names" окно показов и экспортируют имена только до первого неалфавитного символа

С короткими названиями , экспорт "Power-left" и "Power-1" оба имеют схожесть в названии "Power", который позволяет многократному экспорту с теми же самыми функциональными возможностями, но различными {именами}, которые будут отображены, одним именем. Чтобы удалять дисплей входа{контакта} полностью, используйте позицию табуляции "Layers" побочной области (см Section 4-5). в этой панели - варианты, чтобы делать текст экспорта полностью невидимым.

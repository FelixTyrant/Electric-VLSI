При работе с Electric инкрементное проверочное устройство проектирования (DRC) всегда запущены. Это делается для постоянного контроля выполняемых действий и для вывода сообщений об ошибках, при их обнаружении. При работе с топологией элементов, DRC предназначено для обнаружения геометрических ошибок проектирования.

Настройки DRC находятся в свойствах на закладке "DRC" меню (File / Preferences..., "Tools" , "DRC").

По-умолчанию этот инструмент всегда включен. Для выключения, уберите отметку "On" напротив "Incremental DRC".

В технологии MOS контактные узлы, при увеличении их размера, автоматически увеличивают количество контактных площадок. (подробнее Section 7-4-1). По-этому, из-за обработки каждой контактной площадки в отдельности, большие контактные узлы вызывают сложности у DRC устройства. Для экономии времени установите пункт "Ignore center cuts in large contacts", при этом DRC будет обрабатывать только площадки лежащие по контуру узла.

DRC проектные нормы для других технологий могут требовать особых правил, которые будут приводить к большим временным затратам. Игнорировать эти ошибки можно, установив пункт "Ignore area checking" ( для минимальных требований) и "Ignore extension rules" (для специальных требований).

После проведения анализа схемы, просмотрите сообщения об ошибках на закладке explorer (подробнее Section 4-8).

В нижней части экрана расположено окно сообщений. В окне сообщений отображается листинг многих команд, и сообщения об ошибках.

Текст сообщений может корректироваться, используя команды меню (Edit):Cut, Copy, и Paste. В меню (Window / Messages Window) вы можете удалить весь текст командой Clear. Также для этих операций можно использовать right-click. С помощью Cut, Copy можно копировать текст в окно сообщений.

Текст сообщения сохраняется с помощью команды меню (Window / Messages Window). Чтобы вывести окно сообщений на передний план используется командаSet Font....

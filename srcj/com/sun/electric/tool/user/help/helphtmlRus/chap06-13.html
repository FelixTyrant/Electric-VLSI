Electric использует отдельные части Java для всех действий. Из-за этого, если система сталкивается с ошибкой, это прерывает часть, но главная программа продолжает выполняться.

Если данная часть содержит ошибку и выполнение продолжется, то Вы не будете способны запустить другие команды.(см. Раздел 4-8 ). Даже Quit команда может не выполниться. Чтобы решить эту проблему, используйте Force Quit (and Save) команда (в меню File ).

Если Вы подозреваете, что база данных испорчена, используйте команды подсистемы Check Libraries команда (в меню File ). Check команда исследует базу данных, но не устанавливает ошибки. Repair команда проверяет и восстанавливает базу данных.

Сети должны быть пронумерованы. Сделайте это с Redo Network Numbering команда (в меню Tool / Network).

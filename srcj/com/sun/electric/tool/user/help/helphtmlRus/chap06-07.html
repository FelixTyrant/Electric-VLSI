Electric имеет механизм отмены, который прослеживает все изменения, сделанные в течение сеанса. Когда команда запущена, она и ее побочные эффекты сохранены

Команда Undo (в меню Edit) полностью отменяет последнее сделанное изменение (это включает любые изменения, которые возможно, были сделаны другими инструментальными средствами). Множественное использование команды Undo продолжает отменять далее . Команда Redo восстанавливает изменения, до самого последнего сделанного изменения.

Вы можете также использовать отмену (против часовой стрелки) и восстановление (по часовой стрелке) с помощью иконок на панели инструментов.

Electric хранит только последние 40 изменений, так что все более старшее не может быть отменено. Чтобы увеличиваться количество изменений, которое будет сохраняться, используйте опцию"General" (в меню File / Preferences..., раздел "General", вкладку ⌠General■), и измените поле "Maximum undo history". Для того чтобы просмотреть хронологию изменений, которые были сделаны, используйте команду Show Undo List (в меню Edit / Info).

В Electric▓е, почти каждая команда является обратимой, но есть некоторые исключения. Команды, которые записывают файлы на диск, не являются обратимыми, потому что Electric не может удалить файл с диска. Также команды, которые делают обширные изменения (типа библиотечного удаления), не являются обратимыми.

Другая полезная команда для того, чтобы управлять сделанными изменениями √ Repeat Last Action (в меню Edit). Она повторяет последнюю команду, но только для команд, которые могут быть осмысленно повторены

Graphically, Entire Library команда (в меню Edit / Cell Info) отображает график каждого элемента в библиотеке. The Graphically, From Current Cell команда отображает график элемента, который размещается наверху. Эти команды создают график иерархии элемента. Этот график - фактически новый элемент, названный "CellStructure", сформированный из точек разветвления Фотошаблона, и сохраненный в текущей библиотеке.

Чтобы изменять заданный по умолчанию размер и начало выделения всего нового текста, используйте опцию "Text" (в меню File / Preferences ..., раздел "Display, вкладка ⌠Text■). Главная часть диалогового окна определяет, какие типы текста будут изменяться. Средняя часть позволяет Вам управлять, как тип текста изменится в будущем. Вы можете установить размер, начало выделения, шрифт, и стиль.

Ниже всплывающий раздел "Default Text Style", который устанавливает заданный по умолчанию шрифт для Electric▓а. Ниже этого - установка для ⌠Global text scale". Обычно, весь текст показан в 100 % от его заявленного размера. Однако, Вы можете масштабировать весь текст, печатая другое значение кроме 100 в этом поле. Вы можете также использовать команды Increase All Text Size Размер и Decrease All Text Size (в меню Edit / Text), чтобы изменить это значение, и изменить размер всего отображенного текста.

 


Нижняя часть диалогового окна управляет "smart placement" текста, который корректирует точку захвата согласно среде текста. В настоящее время это применяется только, чтобы экспортировать имена, которые помещены относительно провода, соединяющегося с экспортируемым узлом. Например, если узел на левом конце провода имеет экспорт, и "Horizontal" размещение установлено во "Inside", то экспортный текст присоединится на левой стороне, заставляя метку появиться в проводе.

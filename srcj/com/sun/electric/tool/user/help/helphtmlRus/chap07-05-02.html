Многостраничные схемотехнические ячейки представлены в Electric с помощью page map, которая существует на каждой области схемотехнической ячейки. Для того, чтобы создать такую многостраничную ячейку? используйте команду Make Cell Multi-Page (в меню Cell / Multi-Page Cells). После этого вы обнаружите себя редактирующим первую страницу многостраничной ячейки.

Можно добавлять страницы к текущей многостраничной ячейке с помощью команды Create New Page (в меню Cell / Multi-Page Cells). Удалить текущую странцу можно удалить с помощью Delete This Page. Чтобы перейти на следующую страницу, следует использовать Edit Next Page.

Старые версии Electric имели несколько способов вывода многостраничных документов. Если же система выводит документы не правильно, прибегая к этим способом, используйте команду Convert old-style Multi-Page Schematics.

Рамки являются некоторым графическим вспомогательным инструментом, они могут быть отображены с помощью команды Cell Properties... (в меню Cell). Многостраничные ячейки требуют рамку на каждой странице этой ячейки, но присутствие рамок опционально в прочих ячейках.

Размер рамки бывает "Half-A", "A", "B", "C", "D", и "E". Рамка может быть горизонтальной (landscape) или вертикальной (portrait). Можно выводить титульную рамку в правом нижнем углу. Имя разработчика также может быть задано для каждой ячейки.

Кроме имени разработчика, ячейки с рамкой содержат название компании и название проекта. Эти значения задаются не для каждой ячейки, но их можно задать для каждого пользователя. Индивидуальные библиотеки также могут содержать в себе такую информацию.

Панель "Frame" (в меню File / Preferences..., раздел "Display", панель "Frame") позволяет вам задавать и изменять все эти значения. Имя разработчика берется сначала из ячейки, потом из библиотеки (если такового не указано в ячейке) и, наконец, из умолчаний системы (если не указано ни в ячейке, ни в библиотеке).

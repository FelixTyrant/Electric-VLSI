SUE (Schematic User Environment) - формат баз данных схемотехнического редактора SUE от компании Micro Magic. Прочая информация содержится в Разделе 3-9-2. Опции SUE управляются с панели "SUE" (меню File / Preferences..., раздел "I/O", панель "SUE").

Это диалоговое окно управляет тем, будут ли транзисторы иметь стандартную 3-терминальную конфигурацию или же 4-портовую конфигурацию с подложкой.

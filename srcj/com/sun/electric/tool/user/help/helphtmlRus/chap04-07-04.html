Для нахождения расстояния между любыми точками рабочего поля, используют команду "Measure" из панели инструментов.

Также можно использовать команду Toggle Measure Distance меню (Edit / Modes / Edit).

В режиме измерения каждый клик показывает расстояние от предыдущего.

Очень полезная команда Array... меню (Edit), которая позволяет продублировать, выделенные объекты. (подробнее..... Section 6-4).

Electric написан на Java, и распространяется как едиственный файл с расширением ⌠.jar■, обычно именуемый ⌠electric.jar■. Существует две разновидности ⌠.jar■ файлов: с исходным кодом и без него. Оба файла запускают Electric, но файл с исходным кодом более объемный, потому что содержит Java код.

Electric требует Java версии 1.3 или более высокую. Он был протестирован на Java версии 1.5.

Если Вы извлекаете исходный код из ".jar" файла и построить Electric, обратите внимание, что есть некоторые проблемы при сборке для Macintosh OS/X .

    .

Управление Памяти

Одна проблема с Java состоит в том, что Java Виртуальная Машина имеет предел памяти. Этот предел препятствует программам становиться слишком большими, и ускоряет сборку "мусора". Однако, это препятствует редактированию больших схем. Если Electric требует для работы больший объем памяти, Вы можете запросить, чтобы соответственно использовалось больше оперативной памяти. Чтобы сделать это, используйте "Общее" предпочтение (в меню File / Предпочтение ..., "Общий" раздел ", Общая" вкладка{позиция табуляции}). Внизу диалога - память, ограничивают поле. Для вступлений в действие изменений, необходимо перезапустить Electric.

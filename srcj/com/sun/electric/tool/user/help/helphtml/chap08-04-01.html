The Simulation (Spice), Simulation (Verilog), and Simulation (Others) commands of the Tool menu are able to produce input specifications for a number of different simulators. All these commands work on the current cell and require that all named points be exports. It is also necessary to have power and ground exports.

Electric can write netlists for these simulators:

SimulatorLevelNetlist Command
SpicecircuitTool / Simulation (Spice) / Write Spice Deck...
CDLcircuitTool / Simulation (Spice) / Write CDL Deck...
VerilogfunctionalTool / Simulation (Verilog) / Write Verilog Deck...
IRSIMswitch-levelTool / Simulation (Others) / Write IRSIM Deck...
MaxwellcircuitTool / Simulation (Others) / Write Maxwell Deck...

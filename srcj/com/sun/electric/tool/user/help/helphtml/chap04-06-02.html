The first color control of the Preferences... command is in the "Colors" tab of the "Display" section. The top half of the dialog lists all possible things that can have their color chaned:
  • The transparent colors (all layers associated with a transparent layer are listed).
  • The opaque layers.
  • Special colors such as the background, text, etc.

When changing the background color, note that it must contrast with both the highlight color and the inverse of the highlight color (the inverse is black in the default settings).

Electric has three different color schemes which have predefined colors. The subcommands of the Color Schemes command of the Window menu let you switch between them. With the Black Background Colors subcommand, the background becomes black; with the White Background Colors subcommand, the background becomes white; With the Restore Default Colors subcommand, the background becomes gray (the default).

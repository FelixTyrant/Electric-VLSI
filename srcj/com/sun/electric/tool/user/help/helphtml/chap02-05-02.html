To change the width of an arc, issue the Interactively subcommand of the Size command of the Edit menu. Note that the arc stretches about its center so that an edge is at the cursor location. Click a button to make the change. To change the size of more than one arc at a time, select the arcs and use the All Selected Arcs... subcommand.

Another way to change an arc's width is to select it and use the Object Properties... subcommand of the Properties command of the Edit menu.

The "Easy to Select" checkbox enables selection of the arc with a simple click (see Section 2-1-5).

Note that when typing size amounts into a dialog, specify the size of the highlighted area. A CMOS active arc shows highlighting only on its active area, even though the complete arc has implant regions that are much larger.

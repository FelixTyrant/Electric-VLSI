Before describing the commands for controlling the editing windows, it is useful to mention the messages window, which contains scrollable text. The messages window is the place for all textual information.

The text in the messages window can be selected with the cursor and edited with the Cut, Copy, and Paste commands (in menu Edit). You can remove all text with the Clear command (in menu Window / Messages Window).

The text in the messages window can be saved to disk by using the Save Messages... command (in menu Window / Messages Window). You will be prompted for the place to save the text.

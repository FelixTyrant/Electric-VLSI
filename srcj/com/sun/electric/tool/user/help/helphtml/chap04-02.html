Before describing the commands for controlling the editing windows, it is useful to mention the messages window, which contains scrollable text. The messages window is the place for all textual information.

The text in the messages window can be selected with the cursor and edited with the Cut, Copy, and Paste commands (in menu Edit). You can remove all text with the Clear command (in menu Window / Messages Window). In addition, you can right-click in the messages window to Cut, Copy, Clear, or Paste the selected text. You can also use the right-click context menu to Cut or Copy all of the text in the window.

The text in the messages window can be saved to disk by using the Save Messages... command (in menu Window / Messages Window). You will be prompted for the place to save the text. The messages window can be cleared with the Clear command. You can select the messages window font with the Set Font... command.

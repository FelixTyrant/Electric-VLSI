Before describing the commands for controlling the editing windows, it is useful to mention the messages window, which contains scrollable text. The messages window is the place for all textual information.

The text in the messages window can be selected with the cursor and edited with the Cut, Copy, and Paste commands of the Edit menu. You can remove all text with the Clear subcommand of the Messages Window command of the Window menu.

The text in the messages window can be saved to disk by using the Save Messages... subcommand of the Messages Window command of the Window menu (you will be prompted for the place to save the text).

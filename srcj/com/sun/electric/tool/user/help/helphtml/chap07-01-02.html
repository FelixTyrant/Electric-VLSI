When Electric begins, the status area shows the current "Technology". To work with a different technology, use the popup at the top of the component menu. Once a choice is made, the component menu will be redrawn to show the primitive nodes and arcs in the new technology.

Electric automatically switches the current technology to match the cell being edited. If there are multiple cells being edited from different technologies, this switching can become annoying. To disable automatic technology switching, use the Preferences... command of the File menu, and in the "New Nodes" tab of the "General" section uncheck "Switch technology to match current cell".

To see a list of primitive nodes and arcs in the current technology, use the Describe this Technology subcommand of the Info command of the Edit menu.

Some technologies have settable options that further customize them. The "Technology" tab of the "Technology" section of the Preferences... command of the File menu provides a dialog for controlling those options. More information about this dialog is available from the individual technology sections on MOSIS CMOS (Section 7-4-2), Schematics (Section 7-5-1), and Artwork (Section 7-6-1).

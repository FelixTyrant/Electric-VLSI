Electric is a noun/verb system, meaning that all commands work by first selecting something (the noun) and then doing an operation (the verb). For this reason, selection is important.

Selection (and movement) are done in "selection" mode, which is the default mode when the selection icon is selected.

Selection is done with clicks of the left button. Individual nodes and arcs are selected by clicking over them. Once selected, they are highlighted on the screen. If you use the shift-left button, unhighlighted nodes and arcs are added to the selection, but objects that are already highlighted become deselected.

There are often multiple objects under the cursor (for example, in the area where an arc overlaps a node). To get the object you want, hold the control key while clicking. The control-left button cycles through all objects under the cursor.

The notion of toggling selection (shift-left) and cycling through what is under the cursor (control-left) can be combined. If there are multiple objects under the cursor, and you are trying to toggle the selection, use the control-shift-left button to cycle through them .

To select an object by its name, use the Select Object... subcommand of the Selection command of the Edit menu. The resulting dialog lets you select nodes, arcs, exports, or networks in the cell.

To select everything in the cell, use the Select All subcommand of the Selection command of the Edit menu. To deselect everything, use Select Nothing.

To select everything in the cell that is the same as the currently selected objects, use the Select All Like This subcommand of the Selection command of the Edit menu. The Deselect All Arcs command deselects all selected arcs. This is useful when you wish to select a set of nodes, but you have selected the entire area, including nodes and arcs.
